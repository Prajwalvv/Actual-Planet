questions

What happens when I run out of credits?
Your API calls will return a 429 status. You can upgrade your plan or wait for the monthly reset. We never charge overage fees on the Free tier.
Can I change models per request?
Yes. Each API call includes a model parameter. You can use discover for one query and precise for another. Credits are deducted based on which model you use.
Is this just for financial data?
No. Rylvo is domain-agnostic. The ants don't know what they're monitoring. Configure your swarm with any seed URLs and the agents will discover, validate, and report on whatever domain you point them at.
How does this compare to RAG?
RAG retrieves static documents. Rylvo deploys live agents that continuously sense, validate, and cross-reference real-time data. The output is pre-processed intelligence with conviction scores, not raw text chunks.
Do you support self-hosting?
Enterprise plans include the option to run a dedicated swarm cluster in your own infrastructure. Contact sales for details.
